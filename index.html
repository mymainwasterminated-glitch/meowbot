MeowBot Privacy Policy

Last Updated: July 11, 2026

Hey there. Here is the breakdown of what data MeowBot interacts with, how it’s handled, and your rights regarding it. Because MeowBot is a moderation tool, it needs access to certain information to actually do its job (like logging actions and keeping track of server rules).

What data does MeowBot see and save?

  • IDs: The bot stores Discord User IDs, Server IDs, Channel IDs, and Role IDs. This is required so the bot knows who is a moderator, which server settings belong to whom, and who has active infractions (warns, mutes, bans).
  • Messages: The bot processes message content in real-time to check for things like spam, blacklisted words, or active commands. We do not save your chat logs to our database. Messages are processed instantly and dropped, unless they trigger a moderation action that needs to be posted to your server's designated staff log channel.
  • Mod History: The bot saves cases generated by moderation actions (e.g., "User X was warned by Mod Y for reason Z on this date"). This stays in the database so your staff team can pull up user histories.

Data sharing

We don't sell, trade, or share any server data with third parties. Period. Everything stays strictly within the bot's secure database and Discord's own ecosystem.

Getting your data deleted

  • If you kick the bot: If MeowBot is removed from a server, its configuration and mod logs are kept for 30 days just in case it was an accidental kick. After that, the server data is cleared out.
  • Manual requests: If you are a server owner or an individual user and want your data wiped from our database immediately, just join the support server or message the developer directly, and we will purge it manually.
